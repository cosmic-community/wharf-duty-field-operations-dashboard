# Deployment Guide

## Vercel (Recommended)

1. Push your code to GitHub
2. Import repository in Vercel
3. Add environment variables:
   - `COSMIC_BUCKET_SLUG`
   - `COSMIC_READ_KEY`
   - `COSMIC_WRITE_KEY`
4. Deploy!

## Other Platforms

This Next.js 16 application can be deployed to:
- Netlify
- AWS Amplify
- Digital Ocean App Platform
- Railway
- Any Node.js hosting
